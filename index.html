Principios de Seguridad Informática y Ciberseguridad

Lester Rodriguez

Concepto

La seguridad informática y la ciberseguridad son dos ramas de la informática que se utilizan para proteger la información y los sistemas de posibles ataques. se encarga de mantener la confidencialidad de los datos de tu organización y de recuperar el control mediante el uso de recursos de gestión de riesgos en el caso de que se haya producido algún incidente., mientras que la ciberseguridad todos aquellos procedimientos, mecanismos y tecnologías que se emplean para proteger una infraestructura informática, cuyo principal objetivo es preservar el activo más importante de una compañía: la información

>

Antiviruses

Un antivirus es un tipo de software que se utiliza para evitar, buscar, detectar y eliminar virus de una computadora. A continuacion una lista de algunos de los antivirus mas populares

  • Avast
  • Bitdefender
  • McAfee
  • Avira
  • Eset

Firewall

Un firewall es un sistema diseñado para proteger las redes privadas del acceso no autorizado y no verificado en una conexión a Internet. Algunas marcas de software de firewall son:

  1. Sophos
  2. Netgate pfSense
  3. Azure Firewall
  4. Check Point Next Generation Firewalls (NGFWs)
  5. FortiGate NGFW
